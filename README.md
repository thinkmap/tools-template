# tools-curl
docker curl wget vim tools images
Alpine-based image with just curl

## Usage

```console
$ docker run --rm tools/curl -fsSL https://www.google.com/
```

## Tags

* `tools/curl:latest`: based on `alpine:latest`
* `tools/curl:edge`: based on `alpine:edge`
